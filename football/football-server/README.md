## Tecnologies  used 
-	Docker
-	NestJS 
-	MariaDB
-   MySQL
-   TypeORM
-   Typescript

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development watch mode
$ npm run start:dev --> to start the container and nest project
$ npm run stop:dev  --> to stop the container and nest project

```
## Test

```bash
# unit tests
$ npm run test:dev

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
